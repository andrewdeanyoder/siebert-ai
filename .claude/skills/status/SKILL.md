# Status

Summarize uncommitted changes in the repo.

## Usage
/status

## Instructions
1. Run `git status --short` to see changed files
2. Run `git diff --stat` to see line changes
3. Present a concise summary table with:
   - File name
   - Type of change (new, modified, deleted)
   - Brief description of what changed
